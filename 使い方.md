# API リファレンス

## TerminationCondition クラス

最適化の終了条件を記述する。

| パラメータ | デフォルト | 説明 |
|---|---|---|
| `maxIter` | `INT_MAX` | 最大世代数 |
| `timeLimit` | `DBL_MAX` | 実行時間の上限（秒） |
| `xtol` | `0` | パラメータ空間における最適解の近傍の大きさの基準値（DE では無効） |
| `ftolAbs` | `0` | 目的関数の改善率の基準値 |
| `ftolRel` | `0` | 目的関数の改善率の基準値 |
| `targetError` | `0` | 目的関数が目標値を下回ったときに終了 |
| `stall` | `INT_MAX` | 停滞期間の基準値（世代数）。`\|bestSoFar - currentBest\| < ftolAbs + ftolRel * bestSoFar` の状態が stall 世代続いたら終了 |

### `run_de`

```python
def run_de(
    pop_size: int,
    terminationCondition: Mapping[str, Any],
    lower_lim: float = 1e-3,
    upper_lim: float = 1e4,
    seed: int = 1,
) -> list[Any]
```

`[lower_lim, upper_lim)` の間で指数分布に基づいてランダムに速度定数の初期値を生成し、差分進化法を実行する。

**引数**
- `pop_size` — 速度定数の組の数（`>= 3`）
- `terminationCondition` — 最適化の終了条件
- `lower_lim` — ランダム生成される速度定数の下限
- `upper_lim` — ランダム生成される速度定数の上限
- `seed` — シード値

**返り値**
- 最適化終了後の速度定数の組の配列（残差平方和の小さいものから昇順）

### `simulate`

```python
def simulate(
    t: Sequence[float],
    constant: Sequence[float],
    reaction_ids: Iterable[int],
) -> Any
```

与えられた各 `t` における化学種の濃度、および反応の進行度を求める。

**引数**
- `t` — 追跡する時点
- `constant` — 速度定数の組
- `reaction_ids` — 追跡する反応の反応 ID

**返り値** `ReactionProgressResult`
- `timePoints` — `t` の要素数
- `t` — 追跡を行った時点
- `y` — `y[species][time_index]` = `t[time_index]` 分後における `species` 番目の化学種の濃度（二次元配列）
- `reactionProgress`
  - `reaction_ids` — 追跡した反応 ID
  - `J` — `J[reaction_id][time_index]` = `t[time_index]` 分後における `reaction_id` 番目の反応進行度（二次元配列）
  - `reaction_labels` — `reaction_ids` 番目の反応のラベル（例: `"1 + 2 -> 3 + 4"`）
