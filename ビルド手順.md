# 開発者向けローカルビルド手順

C++ 拡張を手元でビルドして動作確認する場合の手順。`pip install` のみ行う場合は不要。

すべて Project root で実行する。SuiteSparse → SUNDIALS → 本体の順でビルドすること。

---

## 1. SuiteSparse のビルド・インストール

```bash
cmake -S extern/SuiteSparse -B build/deps/build/suitesparse \
  -DCMAKE_INSTALL_PREFIX=$PWD/build/deps/install/suitesparse \
  -DSUITESPARSE_ENABLE_PROJECTS="klu;amd;colamd;btf" \
  -DSUITESPARSE_USE_FORTRAN=OFF

cmake --build build/deps/build/suitesparse --config Release

cmake --install build/deps/build/suitesparse
```

## 2. SUNDIALS のビルド・インストール

```bash
cmake -S extern/sundials -B build/deps/build/sundials \
  -DCMAKE_INSTALL_PREFIX=$PWD/build/deps/install/sundials \
  -DSUNDIALS_ENABLE_KLU=ON \
  -DKLU_ROOT=$PWD/build/deps/install/sundials

cmake --build build/deps/build/sundials --config Release

cmake --install build/deps/build/sundials
```

## 3. プロジェクトのビルド

```bash
cmake -S . -B build

cmake --build build -j

cmake --install build --prefix "$PWD"
```
